# 技能五子棋

基于 Vue 3 + TypeScript + Vite 的技能五子棋网页游戏。支持双人单机、单人 vs 本地 AI，规则参考喜人奇妙夜 2《技能五子棋》。

## 运行

```bash
npm install
npm run dev
```

浏览器打开终端显示的本地地址（如 `http://localhost:5173`）。

## 构建

```bash
npm run build
npm run preview
```

## 规则摘要

- 标准 15×15 五子棋，先五连者赢得该小局；三局两胜。
- 每回合二选一：**落子**（得 1 技能点）或 **抽技能卡**（消耗 3 技能点，抽到后必须当场使用）。
- 技能卡：飞沙走石、静如止水、力拔山兮、调你离山、两极反转、See You Again、大获全胜、滴水穿石、满盘皆输（等概率）。
- 先手：第 1 小局黑先，后续小局由上局胜者先手。和棋加赛一轮（本局重开）。

## 项目结构

- `src/views/`：标题页、游戏页、结算页
- `src/components/`：棋盘、玩家信息、技能卡堆、确认栏
- `src/game/`：棋盘与判赢、技能效果、对局与回合（纯 TS，可复用到小程序）
- `src/ai/`：本地 AI（启发式 + 落子/抽卡决策）
- `src/stores/`：Pinia 游戏状态

联机模式与微信小程序为预留扩展，当前仅实现单机与 vs AI。
